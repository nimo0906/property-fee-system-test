# 本地版安全边界说明

本系统定位为 Windows / macOS 本地桌面收费系统，默认只监听 `127.0.0.1`，不应直接暴露到公网或未受控局域网。

## 发布默认值

- 服务地址：`127.0.0.1`
- 后台 Cookie：`HttpOnly; SameSite=Lax`
- 业主端 Cookie：`HttpOnly; SameSite=Lax`
- 数据库与备份目录：本机文件系统
- 真实导入 Excel：只保留在本机 `真实数据/`，不提交到 Git

## 禁止事项

1. 不要把 `PM_HOST` 配成 `0.0.0.0` 后直接对外开放。
2. 不要提交 `.env`、`property.db`、`backups/`、真实客户 Excel。
3. 不要继续使用默认 `admin/admin123` 做正式收费。
4. 不要让非管理员执行删除、备份恢复、账号、费率、系统健康修复等操作。

## 若必须局域网访问

如果现场必须从其他电脑访问，需要先补齐：

1. 反向代理或 HTTPS 终止。
2. Secure Cookie。
3. CSRF 防护。
4. 独立管理员密码和岗位账号。
5. 固定备份策略和恢复演练。
