# 物业管理收费系统 macOS 试用说明

## 一、发送给试用客户的话术

您好，附件是「物业管理收费系统」macOS 试用包。请按下面步骤打开：

1. 解压 `物业管理收费系统-v2.0-macos.zip`。
2. 打开解压后的文件夹。
3. 双击 `物业管理收费系统.app`。
4. 如果 macOS 提示无法打开，请进入「系统设置 → 隐私与安全性」，在安全提示处点击「仍要打开」。
5. 系统启动后会自动打开本机桌面窗口和登录页面。

默认后台账号：

```text
用户名：admin
密码：admin123
```

首次正式使用前，请先进入「操作员管理」修改管理员密码。

## 二、试用前提醒

- 本试用包为免安装本地桌面版，不需要联网服务器。
- 数据默认保存在当前 Mac 用户目录，不会上传到云端。
- 本版本未做 Apple 签名和公证，所以 macOS 首次打开可能会拦截，这是正常现象。
- 请不要直接删除 App 运行中的数据目录；如需空白试用，请先关闭 App，再使用交付包里的清空脚本。

## 三、建议试用流程

请客户按下面顺序体验：

1. 登录后台首页，查看收费工作台。
2. 进入「房间管理」「业主管理」，确认基础资料字段。
3. 进入「收费项目」，查看物业费、水费、电梯费、公摊等规则。
4. 进入「抄表管理」，录入水费/电费读数。
5. 进入「物业收费」或「自动出账」，生成账单。
6. 进入「账单管理」，查看账单列表、筛选、详情和打印。
7. 进入「缴费记录」，体验收款登记、收据和打印。
8. 进入「发票管理」，查看已缴账单开票流程。
9. 进入「催缴管理」，查看欠费对象和催缴打印。
10. 进入「对账报表」，查看收入统计和对账信息。
11. 进入「数据备份」，确认手动备份和恢复入口。

## 五、数据保存位置

macOS 默认数据目录：

```text
~/Library/Application Support/PropertyFeeSystemData
```

主要目录：

- `database/property.db`：本机业务数据库。
- `backups/`：自动备份和手动备份目录。
- `imports/`：导入缓存目录。
- `exports/`：导出文件目录。
- `updates/`：系统更新文件目录。
- `logs/`：启动和运行日志。

## 六、常见问题

### 1. 双击 App 没反应怎么办？

先确认是否被 macOS 安全机制拦截。进入「系统设置 → 隐私与安全性」，找到刚才的安全提示，点击「仍要打开」。

如果仍无法打开，把以下信息发给维护人员：

- macOS 版本。
- 是否出现安全提示。
- 解压后的完整文件夹截图。
- `~/Library/Application Support/PropertyFeeSystemData/logs/startup_error.log` 内容。

### 2. 为什么打开后看到旧数据？

因为这台 Mac 之前运行过系统，旧数据保存在本机用户目录。交付包本身不包含本机数据库。

需要空白试用时，先关闭 App，再运行交付包里的：

```text
清空本机试用数据.command
```

### 3. 清空数据会不会直接丢失？

清空脚本会先备份到桌面：

```text
~/Desktop/PropertyFeeSystemDataBackups/
```

再清空本机试用数据目录。

### 4. 试用包可以直接正式使用吗？

可以用于内部试用和流程验证。正式投入使用前，建议完成：

- 真实客户数据导入验收。
- 真实收费流程验收。
- 管理员密码修改。
- 数据备份策略确认。
- Windows 端实机打包验收。
- 如需对外分发，再考虑 Apple 签名、公证或安装器。
