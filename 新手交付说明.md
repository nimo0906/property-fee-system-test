# 物业管理收费系统新手交付说明

## 一、软件怎么打开

### Windows

1. 解压交付包。
2. 打开解压后的文件夹。
3. 双击 `PropertyFeeSystem.exe`。
4. 如果 Windows 出现安全提示，点击「更多信息」再点击「仍要运行」。
5. 浏览器自动打开后登录系统。

### macOS

1. 解压交付包。
2. 双击 `物业管理收费系统.app`。
3. 如果 macOS 拦截，进入「系统设置 → 隐私与安全性」，点击「仍要打开」。
4. 浏览器自动打开后登录系统。

默认账号：

```text
用户名：admin
密码：admin123
```

正式使用前必须进入「操作员管理」修改默认密码。

## 二、第一次正式使用顺序

1. 进入「操作员管理」，修改 `admin` 密码。
2. 按岗位创建账号：业务管理员、财务、客服业务编辑、管理层只读。
3. 进入「数据备份」，手动创建一次备份。
4. 进入「数据导入」，导入业主、房间、合同等基础资料。
5. 进入「收费项目」，核对物业费、水费、电梯费、公摊、商业收费项目。
6. 进入「账单管理 → 生成账单」，先做生成前核对，再确认生成。
7. 进入「账单管理」核对本批次金额和欠费状态。
8. 进入账单详情或缴费入口登记收款。
9. 打印收据。
10. 进入「对账报表」导出欠费、收款、减免和客户汇总。
11. 每天下班前进入「数据备份」确认最近备份时间。

## 三、数据保存在哪里

Windows 默认数据目录：

```text
%APPDATA%\PropertyFeeSystemData
```

macOS 默认数据目录：

```text
~/Library/Application Support/PropertyFeeSystemData
```

主要文件和目录：

| 名称 | 作用 |
|---|---|
| `database/property.db` | 业务数据库 |
| `backups/` | 自动备份和手动备份 |
| `imports/` | 导入过程文件 |
| `exports/` | 导出文件 |
| `logs/` | 启动和错误日志 |

不要手动删除 `property.db`。需要迁移或备份时，优先使用系统内「数据备份」功能。

## 四、备份和恢复

### 什么时候备份

- 第一次导入真实资料前。
- 批量导入资料前。
- 批量生成账单前。
- 批量修正金额前。
- 收款高峰结束后。
- 每天下班前。
- 恢复备份前。

### 怎么备份

进入「数据备份」，点击「创建备份」。

页面会显示：

- 最近一次备份。
- 备份目录。
- 备份文件列表。

### 怎么恢复

1. 进入「数据备份」。
2. 找到要恢复的备份。
3. 点击「恢复」。
4. 核对备份摘要。
5. 输入「恢复」。
6. 点击「确认恢复」。

恢复会覆盖当前数据库。系统会先自动创建一份 `auto_before_restore_*.db`，方便发现恢复错了以后再回退。

## 五、日常收费流程

### 出账

1. 进入「账单管理 → 生成账单」。
2. 选择日期、楼栋、房间范围和收费项目。
3. 点击「生成前核对」。
4. 核对涉及房间、收费项目、金额合计、异常核对清单。
5. 确认无误后点击「确认生成账单」。

如果看到无业主、面积为 0、抄表类无读数等异常，先修正基础资料或抄表数据，不要直接收费。

### 收款

1. 进入「账单管理」。
2. 按日期、楼栋、客户或批次筛选账单。
3. 展开分组核对收费项目。
4. 点击「缴费」。
5. 核对本次可收金额、历史已收和当前欠费。
6. 确认收款。
7. 打印收据。

已结清账单不能重复收款。

### 对账

进入「对账报表」，按日期和楼栋筛选后，可导出：

- 对账 CSV。
- 欠费明细 CSV。
- 收款汇总 CSV。
- 收款流水 CSV。
- 减免明细 CSV。
- 客户汇总 CSV。
- 租户欠费排行 CSV。
- 费用欠费 CSV。
- 经营分析 Excel。

对账金额口径：

| 指标 | 含义 |
|---|---|
| 应收合计 | 当前筛选范围内账单金额合计 |
| 已收合计 | 当前筛选范围内实际收款合计 |
| 欠费合计 | 应收合计减已收合计 |
| 减免合计 | 已做金额调减的差额合计 |

## 六、换电脑或多人共用

本系统是本地桌面版。正式多人共用时，不建议每个人各自打开一份程序再共用同一个 SQLite 数据库文件。

推荐做法：

1. 指定一台电脑作为收费主机。
2. 只在这台电脑上运行系统。
3. 其他电脑通过浏览器访问这台电脑的局域网地址。
4. 每天在主机上做备份。

如果只是换一台电脑继续使用：

1. 在旧电脑「数据备份」创建备份。
2. 把备份文件拷贝到新电脑。
3. 在新电脑打开系统。
4. 进入「数据备份」恢复该备份。
5. 核对房间、账单、收款、报表金额。

## 七、出问题时发什么给维护人员

请发送以下信息：

1. 使用的是 Windows 还是 macOS。
2. 软件包文件夹截图。
3. 当前页面截图。
4. 操作步骤：点击了哪里、输入了什么、出现什么结果。
5. 数据目录里的日志文件：

```text
logs/startup_error.log
logs/request_error.log
logs/runtime_status.log
```

如果是 Windows，可以先运行：

```text
diagnose_windows.bat
```

把窗口内容发给维护人员。

## 八、不能随便做的事

- 不要删除数据目录。
- 不要直接改 `property.db`。
- 不要把真实数据包发给无关人员。
- 不要在未备份前做批量导入、批量修正、恢复备份。
- 不要对已经结清的账单重复收费。
- 不要在金额异常时直接打印收据。
