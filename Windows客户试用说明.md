# 物业管理收费系统 Windows 试用说明

## 一、普通用户怎么打开

收到 Windows 试用包后，只需要按下面步骤操作：

1. 解压 `物业管理收费系统-v2.0-windows.zip`。
2. 打开解压后的文件夹。
3. 双击：

```text
PropertyFeeSystem.exe
```

4. 如果 Windows 安全中心或 Defender 提示风险，请选择「更多信息」→「仍要运行」。
5. 程序启动后会自动打开浏览器页面。

默认后台账号：

```text
用户名：admin
密码：admin123
```

正式使用前请修改默认密码。内部试用可以先用默认账号体验流程。

## 二、如果看到旧测试数据

这说明这台电脑之前运行过系统，旧数据保存在当前 Windows 用户目录：

```text
%APPDATA%\PropertyFeeSystem
```

这不是交付包混入了旧数据库。

需要空白试用时：

1. 先关闭 `PropertyFeeSystem.exe`。
2. 双击交付包里的：

```text
清空本机试用数据.bat
```

脚本会先把旧数据备份到桌面：

```text
Desktop\PropertyFeeSystemDataBackups
```

然后清空本机试用数据。重新打开 `PropertyFeeSystem.exe` 后，会生成新的空白数据库。

## 五、常见问题

### 1. 双击 exe 没反应怎么办？

先运行交付包里的：

```text
diagnose_windows.bat
```

把窗口内容发给维护人员。

### 2. 提示缺少依赖怎么办？

正常交付包不需要用户安装 Python。只有源码版或打包人员才需要运行：

```text
install_windows_dependencies.bat
```

### 3. 数据在哪里？

默认在：

```text
%APPDATA%\PropertyFeeSystem
```

主要文件：

- `property.db`：业务数据库。
- `backups`：自动备份目录。
- `startup_error.log`：启动失败日志。
- `runtime_status.log`：诊断信息。
